# A ruby example of Monobank API webhook sign verification
An example of Ruby code of Monobank API webhook sign verification.

## How to run
1. Clone this repo
2. Go to the repo folder
3. Run:
```bash
ruby main.rb
```

The output should be:
```bash
OK
```

## Development
1. Clone this repo
2. Go to the repo folder
3. Run:
```bash
bundle install
```
4. Run:
```bash
bundle exec rspec
```
